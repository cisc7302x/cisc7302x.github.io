    Goals #

    To Build a fully functional general purpose computer from the ground up.

    The course will be divided into two halves:

    1. Hardware stack: nand gate -> assembler

      • Electronic circuits
      • Boolean algebra
      • Logic Gates (digital primitives)
      • Arithmetic Logic Units (combinational circuits)
      • Registers and Memory (sequential circuits)
      • Von Neumann Architecture
      • An Assembler

    2. Software stack: virtual machine -> operating sytem
